mkdeb

The Debian packaging helper

mkdeb aims to provide a convenient way to generate Debian packages from upstream release archives.

Download View on Github

Packages generation is based on collection of pre-defined recipes available in a dedicated Git repository.

Those recipes are simple YAML files that may provide additional maintainers scripts and files to be appended to the resulting Debian package.

See recipes documentation for further details.
